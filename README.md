# trivia

Simple React + Vite sandbox for learning and experimenting with a trivia app.

## Included

- React + Vite setup
- React Router with pages:
  - Home
  - Categories
  - Game
  - Admin
- Simple navigation
- Clean folder structure:
  - `src/pages`
  - `src/components`
  - `src/lib`
  - `src/styles`

## Not included yet

- Trivia questions/data
- Supabase
- Authentication

## Run locally

```bash
npm install
npm run dev
```
