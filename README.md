# trivia

Simple React + Vite sandbox for learning and experimenting with a trivia app.

## Included

- React + Vite setup
- React Router with pages:
  - Home
  - Categories (loads active categories from Supabase)
  - Game
  - Admin
- Simple navigation
- Clean folder structure:
  - `src/pages`
  - `src/components`
  - `src/lib`
  - `src/styles`

## Not included yet

- Trivia questions/data
- Authentication

## Run locally

1. Install dependencies.
2. Copy `.env.example` to `.env`.
3. Start the app.

```bash
npm install
cp .env.example .env
npm run dev
```
